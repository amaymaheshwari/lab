## üìÇ Projects

### 1. [Personal Portfolio](./website)
> **Directory:** `/website`

My primary digital presence. A dynamic, interactive showcase of my work, travel, and interests.

*   **Core Tech**: Flask (Python), Vanilla CSS, JSON-based content.
*   **Key Features**:
    *   **Global Footprint**: Interactive Leaflet.js world map highlighting visited regions.
    *   **"Henry Stickmin" Bot**: A fully articulated SVG character that physically parkours across DOM elements as you scroll (Archived).
    *   **Premium Dark Mode**: Apple-inspired glassmorphism design system.
    *   **Live Sync**: Auto-fetching of latest YouTube videos and music tracks.

### 2. [Leadership Under Constraints](./mba_portfolio)
> **Directory:** `/mba_portfolio`

An executive-style portfolio designed for high-stakes professional narratives.

*   **Core Tech**: Flask (Python), Google Gemini AI, Flat-file CMS.
*   **Key Features**:
    *   **AI-Refined Editor**: Built-in admin panel uses **Google Gemini 2.0 Flash** to rewrite raw thoughts into "Executive Style" memos instantly.
    *   **Case Study System**: Structured data format for presenting complex decision-making scenarios.
    *   **Design**: Minimalist "Swiss Style" typography focused on readability and impact.

### 3. [AI News Bot](./ai_news_bot)
> **Directory:** `/ai_news_bot`

An automated agent that scours the web for the latest AI breakthroughs.

*   **Core Tech**: Python, Web Scraping, SMTP Emailer.
*   **Key Features**:
    *   **Automated Aggregation**: Daily scraping of key tech sources.
    *   **Smart Filtering**: Uses keywords to separate signal from noise.
    *   **Daily Brief**:compiles and sends a structured email summary of the day's top AI news.
